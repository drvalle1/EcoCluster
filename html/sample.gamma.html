sample.gamma {EcoCluster}R Documentation

Sample the TSB prior parameter

Description

This function samples the truncated stick breaking (TSB) prior parameter gamma

Usage

sample.gamma(v, ngroup, gamma.possib)

Arguments

v

vector of length L with probabilities

ngroup

maximum number of location groups (K)

gamma.possib

vector of possible gamma parameter values

Value

this function returns a real number corresponding to gamma

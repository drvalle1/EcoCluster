sample.gamma.u {EcoCluster}R Documentation

Sample gamma.u

Description

Sample the TSB prior parameter for species groups (gamma.u) from its full conditional distribution

Usage

sample.gamma.u(uk, gamma.possib, ngroup.spp)

Arguments

uk

vector of size KS with probabilities

gamma.possib

this is a vector containing the possible values that gamma.u can take

ngroup.spp

maximum number of species groups (KS)

Value

this function returns a real number (gamma.u)

sample.gamma.sam {EcoCluster}R Documentation

Sample gamma

Description

This function generates a sample of the truncated stick breaking prior parameter gamma

Usage

sample.gamma.sam(v, ngroups, gamma.possib)

Arguments

v

vector of size KS with the stick-breaking parameters

ngroups

maximum number of species groups (KS)

gamma.possib

set of possible values for gamma

Value

a sample of gamma

sample.gamma.v {EcoCluster}R Documentation

Sample gamma.v

Description

Sample the TSB prior parameter for location groups (gamma.v) from its full conditional distribution

Usage

sample.gamma.v(vk, gamma.possib, ngroup.loc)

Arguments

vk

vector of size KL with probabilities

gamma.possib

vector containing the possible values that gamma.u can take

ngroup.loc

maximum number of location groups (KL)

Value

this function returns a real number (gamma.v)
